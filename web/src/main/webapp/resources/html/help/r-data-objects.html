Expression Atlas data in R

For differential expression experiments in Expression Atlas, you can download a file containing the expression data and meta-data that you can load into R. Click the button on the top-right of any differential experiment page to download one.

How to load an Atlas experiment summary in R

Start an R session on your computer.

Load the object you downloaded into your R session:

load( "/path/to/E-MEXP-1066-atlasExperimentSummary.Rdata" )

This has created an object called experimentSummary. This object is a SimpleList object. Each element is one of three Bioconductor objects:

  • SummarizedExperiment (RNA-seq experiments)
  • ExpressionSet (1-colour microarray experiments)
  • MAList (2-colour microarray experiments)

How to use it

Seeing what's available

Now that you have loaded it, you can access the objects by their names in the SimpleList. To see these names, type names( experimentSummary ).

The names of objects in a microarray experiment correspond to the ArrayExpress array design accessions used in the experiment. In our example of E-MTAB-1066, only one array design was used, which is usually the case for microarray experiments in Expression Atlas. However, sometimes microarray experiments contain more than one data-set, each of which was done using a different array design. For example, the experiment "Transcription profiling by array of spinal cord from rats with experimental autoimmune encephalomyelitis" uses three different versions of the Affymetrix GeneChip Rat Genome U34 array (A, B, and C). For this experiment, the SimpleList object that you load will have three elements, each one with data from one of the three array versions used.

Expression Atlas SimpleList objects for RNA-seq experiments will always contain only one element, named rnaseq.

Accessing the expression data

First, to keep things simple we assign one of the objects in the SimpleList to a new variable, by typing e.g. mtab1066_affy35 <- mtab1066_list[[ "A-AFFY-35" ]] .
